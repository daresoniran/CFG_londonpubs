Pubs of London

Dashboard – Effective modelling using python on PyCharm

The aim of this dashboard is to present the same data analysis with three different visualisations. All models present the number of pubs in each borough of London.
On each page, we present why we chose the chart, reflect on the positives and negatives and present alternatives for improvement (as we further improve our coding skills).

Keys: Name of pub, Latitude, Longitude, Local Authority (London Borough)
